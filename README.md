# Rule-Based Code Explainer

A beginner-friendly Python project that explains simple Python code using rule-based logic.

## Features
- Accepts Python code as input
- Detects syntax errors
- Detects indentation errors
- Explains common Python constructs such as:
  - print()
  - for loops
  - if statements
  - function definitions
  - variable assignments

## Built With
- Python
- Streamlit
- AST (Abstract Syntax Tree)

## How It Works
1. The user enters Python code
2. The app checks whether the code is valid
3. If the code has errors, it shows an error message
4. If the code is valid, the app explains it using rule-based logic
